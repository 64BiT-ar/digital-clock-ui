# Digital Clock UI

A clean, browser-based digital clock built with HTML, CSS, and JavaScript. The app displays the current time in real time with a full-screen visual layout and a simple, focused interface.

**Author:** Ahmed Raza

## Features

- Live digital clock with second-by-second updates
- 12-hour time format with AM/PM
- Full-screen background image styling
- Lightweight, dependency-free implementation

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Run

1. Open `index.html` in a web browser.
2. The clock will start automatically and update every second.

## Project Structure

- `index.html` - Main page structure
- `style.css` - Visual styling and layout
- `script.js` - Clock logic and time updates

## Description for GitHub

A responsive digital clock web app built with vanilla HTML, CSS, and JavaScript, featuring a live time display and a clean full-screen interface.
